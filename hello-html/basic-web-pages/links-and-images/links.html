Este ejemplo va sobre links e imágenes

Esta página en particular va sobre links! Hay tres tipos de linlks:

  • Links absolutos para dirigir a los usuarios a un sitio web diferente. Como Mozilla Developer Network, el cual es un recurso muy bueno para desarrolladores web.
  • Links relativos, que apuntan a otro archivo en su sitio web desde el punto de vista del archivo que está editando. Como nuestra página de extras.
  • Links relativos a la raíz, son relativos a la "raíz" de todo el sitio web como home page pero por ahora no es util
